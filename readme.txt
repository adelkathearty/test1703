Тестовое задание: Успешный запуск теста

Для успешного запуска теста необходимо убедиться, что у вас установлены следующие версии:

ChromeDriver версии 111
Java версии 19

Если у вас отличается версия браузера Google Chrome, то вы можете скачать соответствующую версию ChromeDriver по ссылке: https://chromedriver.chromium.org/downloads

Также возможно, что при логине с нового IP-адреса будет запрошена капча, в этом случае желательно ввести её один раз и перезапустить тест. 

Для выбора версии браузера для запуска тестов необходимо открыть файл BrowserSettings.java и установить значение поля BROWSER_AND_PLATFORM. В данном файле  представлены три версии: MACOS, MACOSINTEL, WINDOWS.
